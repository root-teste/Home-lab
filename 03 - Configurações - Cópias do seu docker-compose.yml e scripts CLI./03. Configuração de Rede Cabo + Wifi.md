### 1. Primeiro, vamos confirmar se o sistema enxerga a placa wireless:

Bash

```
nmcli device
```

### 2. Ativar o Wi-Fi e Escanear Redes

Bash

```
# Garante que o rádio do Wi-Fi está ligado
sudo nmcli radio wifi on

# Lista as redes disponíveis
nmcli device wifi list
```

### 3. Conectar à sua rede Wi-Fi

Substitua `NOME_DA_SUA_REDE` e `SUA_SENHA` pelos seus dados reais:

Bash

```
sudo nmcli device wifi connect "NOME_DA_SUA_REDE" password "SUA_SENHA"
```

### 4. O ajuste para o IP Fixo no Wi-Fi

Como você já reservou o IP `.90` para o cabo de rede no roteador, o ideal é que o Wi-Fi tenha um IP diferente (ex: `192.168.1.91`) para evitar conflitos se ambos estiverem conectados ao mesmo tempo.

---

### 🛡️ Prioridade de Conexão (Metric)

Se o Samsung estiver com o cabo **E** o Wi-Fi ligados, o Linux precisa saber qual caminho é mais rápido. Queremos que ele use o **Cabo** como preferência e o **Wi-Fi** apenas como reserva.

Bash

```
# Define o cabo como prioridade máxima (métrica menor = prioridade maior)
sudo nmcli connection modify "enp1s0" ipv4.route-metric 100

# Define o Wi-Fi como reserva
sudo nmcli connection modify "NOME_DA_SUA_REDE" ipv4.route-metric 600
```
