
O Docker é a peça fundamental. É ele que vai permitir rodar o **Immich**, o **Nextcloud**, **Agent-Zero** e outros recursos de forma isolada, sem "sujar" o seu Debian com milhares de dependências.

**1. Instalar dependências iniciais:**

Bash

```
sudo apt update
sudo apt install ca-certificates curl gnupg lsb-release -y
```

**2. Adicionar a chave oficial do Docker:**

Bash

```
sudo install -m 0755 -d /etc/apt/keyrings
curl -fsSL https://download.docker.com/linux/debian/gpg | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg
sudo chmod a+r /etc/apt/keyrings/docker.gpg
```

**3. Configurar o repositório:**

Bash

```
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] https://download.docker.com/linux/debian \
  $(. /etc/os-release && echo "$VERSION_CODENAME") stable" | \
  sudo tee /etc/apt/sources.list.d/docker.list > /dev/null
```

**4. Instalar o Docker Engine e o Docker Compose:**

Bash

```
sudo apt update
sudo apt install docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin -y
```

---

### 🛡️ Ajuste de Permissão (Importante)

Por padrão, você precisaria digitar `sudo` para cada comando do Docker. Vamos adicionar o seu usuário `serv` ao grupo do Docker para facilitar sua vida:

Bash

```
sudo usermod -aG docker serv
```

**⚠️ ATENÇÃO:** Para essa mudança de grupo funcionar, você precisa deslogar e logar novamente no SSH.
