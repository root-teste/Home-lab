### Desativar a interface sem deletar (O modo "Seguro")

Antes de deletar tudo, podemos apenas dizer ao Debian para não iniciar a interface gráfica no boot. Isso transforma o sistema em modo texto, mas mantém o GNOME lá caso você precise dele em uma emergência física:

Bash

```
sudo systemctl set-default multi-user.target
sudo reboot
```

_Se um dia quiser o GNOME de volta, é só rodar o mesmo comando com `graphical.target`._

---

### 3. Remover o GNOME Definitivamente (O modo "Server Raiz")

Se você decidir que quer o Samsung 100% dedicado ao NAS e IA, rodaremos estes comandos pelo SSH:

Bash

```
# Remove o GNOME e dependências de interface (Login Manager)
sudo apt purge gnome-shell gnome-session gdm3 desktop-base -y
sudo apt autoremove -y
sudo apt clean
```
